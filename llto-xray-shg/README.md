# LLTO — X-ray second-harmonic generation (SHG)

**Paper:** Woodahl *et al.*, "Probing lithium mobility at a solid electrolyte surface," *Nature Materials* **22** (2023). https://doi.org/10.1038/s41563-023-01535-y

**Data archive (Dryad):** https://doi.org/10.6078/D1N41X

Example SHG simulation of LLTO containing the input file and pseudopotential, using real-time VG-RTTDFT.

For the implementation of the real-time VG-RTTDFT method see
<https://www.sciencedirect.com/science/article/abs/pii/S0010465518300262?via%3Dihub>.
